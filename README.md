# MedLedger-AI (Demo UI)

Simple Flask-based boilerplate UI to demonstrate hospital participation in a federated learning workflow. This project intentionally does NOT include any ML or training logic — it provides routes, pages, and hooks for later integration.

Run:

```bash
python app.py
```

Open http://localhost:5000

Demo credentials:
- admin / adminpass
- hospital1..hospital5 / password

Uploaded datasets are stored in `data/hospitalX/`.

This project is intentionally minimal and ready for extension.
