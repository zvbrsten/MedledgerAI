Detailed Metrics Explanation

Classification Metrics

Accuracy ({{ status.accuracy }})

Overall correctness of the model across all predictions. Represents the percentage of correct predictions out of total predictions.

Precision ({{ status.precision }})

Of all positive predictions made, how many were correct. Important for minimizing false positives in medical diagnosis.

Recall ({{ status.recall }})

Of all actual positive cases, how many did the model correctly identify. Critical for not missing diagnoses.

F1 Score ({{ status.f1_score }})

Harmonic mean of precision and recall. Provides a single metric balancing both false positives and false negatives.

Performance Indicators

AUC-ROC ({{ status.auc_roc }})

Area Under the Receiver Operating Characteristic curve. Measures the model's ability to distinguish between classes across all probability thresholds.

Loss ({{ status.loss }})

Training loss value indicating model error. Lower values indicate better model fit. Decreasing loss over rounds shows improvement.

Training Continuity

As hospitals contribute datasets, metrics are logged continuously. The model improves with each federated learning round, creating an unbroken journey of enhancement.
